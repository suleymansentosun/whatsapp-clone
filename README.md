# WhatsApp Clone

A full-featured WhatsApp clone built with Next.js, NestJS, and MongoDB.

## Features

- Real-time messaging
- User authentication
- Group chats
- Media sharing
- Status updates
- End-to-end encryption
- Push notifications

## Tech Stack

### Frontend
- Next.js
- React
- Socket.io-client
- Material-UI
- Redux

### Backend
- NestJS
- Socket.io
- MongoDB
- JWT Authentication
